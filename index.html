Mis Servicios

Diseño Web

Este servicio consiste en el diseño de landing pages, en que consiste un landing page? es una pagina web estatica donde puedes dar a conocer algun negocio o servicio y se puedan poner en contacto contigo. Para este servicio se hace uso de un servidor para alojar la pagina web, posteriormente se hace el diseño en HTML/CSS y Javascript para el lado del Frontend, el diseño es 100% responsive (adaptado a movil), para que pueda ser consultado en cualquier dispositivo.

Desarrollo Backend

Este servicio consiste en desplegar servidores Linux para la creacion de aplicaciones web de acuerdo a tus necesidades. Este servicio es mas sofisticado ya que se hace uso de herramientas y tecnologias especificas al proyecto, por ejemplo un sistema de ventas, sistema de almacen o cualquier otro tipo sistema.

Wordpress Website

Este servicio consiste en al creacion de sitios web dinamicos o estaticos, con una interfaz amigable al usuario, por ejemplo para la creacion de blogs de noticias o para ofrecer tus servicios en linea, gracias a su interfaz amigable una vez desplegado el sitio web podras hacer ediciones de acuerdo a tus necesidades como agregar Articulos en caso de un blog o editar ciertas Paginas, todo esto accesando con tu usuario y contraseña.
